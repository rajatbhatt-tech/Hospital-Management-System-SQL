# 🏥 Hospital Management System (SQL)

## 🌟 Project Overview
A database-driven analytics project designed to manage hospital operations, track patient admissions, and analyze revenue trends.

## 🔑 Key SQL Features Used:
- **Relational Mapping:** Linked 3 tables using Foreign Keys.
- **Data Integrity:** Used `CHECK` constraints for age validation.
- **Joins & Aggregations:** Used `INNER JOIN`, `GROUP BY`, and `SUM` for business reporting.
- **Automation:** Created `VIEWS` for daily management reports.

## 📊 Business Insights Generated:
1. **Top Performing Doctors:** Based on revenue and patient load.
2. **Patient Demographics:** City-wise distribution.
3. **High-Value Case Tracking:** Identifying surgeries/treatments > ₹50,000.
